Genomic Assembly Overview

This project focuses on the high-fidelity de novo assembly of the Oryza sativa (Rice) genome. Using PacBio HiFi long-reads, we achieved a high-contiguity assembly that serves as a foundation for further functional genomic research and comparative analysis.

Metric Value / Description
Organism Oryza sativa (Kasalath)
Data Type PacBio HiFi (SRR35146894)
Assembler Hifiasm (20 Threads)
Scaffolding RagTag (Reference-guided)
Annotation Augustus 3.5 (Gene Prediction)

Key Findings

The assembly successfully identified nuclear chromosomes along with complete organelle genomes. K-mer profiling (k=21, 51) was used to validate the genome size and heterozygosity levels prior to assembly.
